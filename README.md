# Orbital Motion Simulation
A simple 2D orbital motion simulation written in Python using NumPy and Matplotlib.

## Overview
This project models a small body orbiting a much larger central mass using Newtonian gravity and a time-stepping method.
The default parameters approximate the Earth-Sun system.

## Features
- 2D orbital simulation
- Newtonian gravitational physics
- Numerical time-stepping
- Orbit visualisation using Matplotlib

## How to run
Install the required libraries:

pip install numpy matplotlib

Run the script: 

python Orbital_motion_simulator.py

## Notes
- The orbit is close to circular with the default values
- The simulation uses a simple numerical method, so it's not perfectly accurate
